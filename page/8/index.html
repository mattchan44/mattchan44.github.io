產品雜談05

發表於 2017-06-11 分類於 產品雜談
本文字數： 1.9k 閱讀時長 ≈ 3 分鍾

  今天有時間，報名了參加了傳說中很火熱的人人都是產品經理的線下活動，換個說法，就是傳說中大牛的分享。

  騎共享單車去的，線下活動的地方在五角場財大那邊。路痴，所以找了很久，問了好幾個人才找到地方。我自己原來搞過培訓課程的Sales工作，所以對這種線下課程的東西還是懷有些抵觸情緒。培訓行業做經常做的手段，就是傳說中的線下營銷。一般，會請一些大公司的成功人士，反正外邊看到的都是成功人士。大多數人呢，其實會有莫名的盲從心理，所以我們看到的名利雙收的永遠是被別人請上講臺。同時，這也是圈子裡面經常採用的手段，今天我給你背書，明天你給我背書，因此嘉賓跟機構永遠是各取所需的關係。我前面有講過，產品這個行業其實是兩部分的人合力搞起來的，一方面是大公司的一部分人需要變現的渠道，另外一方面是培訓機構需要新的蛋糕去佔領，去創造價值。早期的關於項目的證書已經爛大街，開發的名聲也不太好，所以需要新的出路。兩方面一拍即合，大家都找到自己的價值。嘉賓呢，成功的點亮了個人的光環，平臺呢，成功圈起來一批的小白供後續的轉化，這都是大批的新鮮資源，可不是電話被打爛掉，被徹底轟炸完，很難挖掘的一批。至於人人後續是否會繼續挖掘剩餘價值呢，不知道，等後面就知道了。

  分享的內容呢，與我而言，其實並沒有發現太多有用的東西，畢竟方法論也好，工作的流程什麼的，這種照搬的結果往往是死的更快。每家面臨的實際情況不一樣，你把聽起來牛逼哄哄的嘉賓放在你家的位置上，可能會做的更好，也更可能做的更差。講到這裡，要說到兩派的人了。一派是這種大公司流程結果下培養出來的螺絲釘，當然也有很多嘉賓有不錯的機遇，能夠耐得住寂寞，所以在三五年之後脫穎而出的。另外一派是野路子出身，這應該是現在大多數產品的出身，各種野蠻生長的公司，所以成功活下來的產品各有各的路子，每個人都形成了自己的獨特認知。當然，這是最簡單粗暴的劃分方法，畢竟現在早過了“人人都是產品經理”的時代。現在是產品爛大街的時代，17年招聘就可見一斑，三年以下經驗的產品已經很難找到非常合適的工作，大家傾向於的是在一個行業內有一定時間的累積，最好是三年左右的產品，因為這樣劃分過來簡單粗暴，而最節省招聘的時間。

  當然，現在關於最基礎的background knowledge的認知，很少有人真的能耐住性子一步一步完成初步的啟蒙，大家都傾向於找到乾貨，快速的轉化成你昇華的養分。想想我當時入行，真的是強行入行，當時光記錄東西的本子就有十幾本。我就是靠著網上質量不高的小白文章完成了我關於產品的初步啟蒙……方法麼，大家都有自己的套路，尤其是到了一定階段之後，因為光從網際網路獲取所謂的乾貨並不能帶給自己任何的提升，到了瓶頸階段，這種時候就需要靠最笨最原始的書籍去嘗試完成突破。當然，這種書籍並不是現在泛濫成災的套路書，七拼八糟一大堆的東西，這一章講使用者體驗，那一章講調研方法，滅掉最原始的引用，就是自己成功的方法論。如果我已經讀過了The Elements of User Experience，我還去讀你那本二手資料的書幹啥。真正的好書是那種，不但給你最核心的結論，還給你連結到原始說法的這種，因為任何思路的來源都不能是自己拍腦袋得出的，總會有一個反覆思索得出結論的過程，而過程是最重要的。這也是所有的成為嘉賓的人敘述中永遠缺失的部分，因為缺乏細節，你永遠無法成功的重現。所有的名人傳記都會記錄被認為是重要因素的坎坷，但是永遠無法知道一些迥異於其他可能導致成功的因素。而即使是名人自己去書寫，也會忽視這樣的要素，因為對於名人而言，這是normal。

  培訓機構就是永遠在做這樣的一件事情。交給你一大堆的資料，然後告訴你讀完就成功了。包括現在的所謂知識付費時代，因為誰也承擔不起手把手教人成功的成本，一則投入跟產出不成正比，二則有失敗的風險。所以知識付費所提供的重要模式就是方法論，其次是Q&A，即付費問答。回到最開始時，我做過一段時間的培訓行業，從好幾年前的行業熟悉人士來講，人人的線下操作手法並不高超，這也是他家需要跟傳統的機構學習借鑑的地方，環節什麼的漏洞太多，缺乏簡單粗暴的轉化手段，或者說從資料角度而言，應該也沒有進行過明顯的增長黑客一類火熱的名詞洗禮吧。

  參與一次就夠了，也總算有了參與傳說中行業內的活動，至於擴展人脈一類的東西其實並沒做到，只有當雙方對等的時候才有成為朋友的可能，才可能建立所謂的人脈。今天的嘉賓，從某種角度而言，是來收割智商稅的，或者建立明星光環的，只要多做幾次活動，人氣有了，財也有了。

關於讀書

發表於 2017-06-06 分類於 詩文隨筆
本文字數： 2.3k 閱讀時長 ≈ 4 分鍾

    許久不寫東西，今天寫點兒關於讀書的事情。從上學開始，貌似一直是處於不斷讀書的過程中，從簡單的識字，到後續的進一步為了淘汰性質的pk，中考，高考決定人生命運的關鍵。想想小學同學，到後面的初中同學，高中同學，最終到大學同學。如果繼續往後走就是碩士同學，博士同學，一層層的篩選，一層層的淘汰，最終大概已經忘記了最初的方向。畢竟，人生之不如意十之八九。愛恨糾葛，自己無法做決定的事情太多，更多的是隨著現實飄轉。

   當然，我是沒太多文化，大學混了幾年畢業之後就出來闖蕩。沒有進過穩定的企事業單位，沒有享受過鐵飯碗的紅利，一直晃盪了好幾年，因為是銷售出身，所以後面一直在一家和另外一家公司之間來回晃悠。一直晃悠到網際網路行業才算初步安定下來，也混成了一枚小產品，打雜的。

    拐回讀書的事情。畢業這幾年也斷斷續續的讀了一部分的書，各種各樣類型的都有吧，其實坦白講，讀書能夠獲得的紅利，還是得基於你後續的認知，後續基於書本內容的再加工。網上也見過一堆再一堆的書單，但是真正能夠讀下去的，其實非常少。走過幾年的路，現在往回看，發現還是得老老實實一步一步走，否則單獨刷那些書沒毛用。其實，雖然到了網際網路時代，但是每個學科的核心知識還是沒有特別大的顛覆，所謂的只是大爆炸充斥的是垃圾資訊。不管是現在的知識付費也好，還是說紅利也好，能夠獲利的這一批人，基本上都是在某一個行業內深耕細作了一定的年月，才能夠轉化足夠的數量。雖然可能載體一直在變化，但是玩的這一批人並沒有特別大的轉換。

    換個角度去看，每個學科總會有一兩本書能夠將核心的內容概括出來，給你最核心的資訊，學科最基礎的認知。所以換句話講，你認真的研讀了這一兩本的教材，已經獲得了最核心的最基礎的認識，剩下的只是補充細節，以及update到最新的學科進展。補充細節需要你去圍繞著核心的知識脈絡去擴展，基礎認知只是涵蓋了學科涉及到的基礎元素，後面每個基礎元素都可能延伸出來無數的書籍，良莠不齊。至於最新的學科進展，則需要去刷論文。畢竟，相對書籍的較長的更新週期而言，最快的肯定是論文，因為論文是隻要有新的知識點就可以隨時重新整理的。還會有一類的論文叫做文獻綜述的說法，簡單講就是根據某個核心論點，延伸出來的相關的觀點。

    相對而言，國內的不少學科的教材還是很有些陳舊，尤其是經濟，金融，心理等等舶來品的教材，很少會有學校直接上原版的教材，這裡面一方面是國內的市場已經被翻譯或者改造的教材佔領，另外一方面則是因為意識形態的緣故，畢竟教材的編者並不會太鳥一個社會主義國家的看法。所以，國內最常見的模式其實是一些教授，將國外教材拼湊一下，出來一部的中國特色的新編著教材，另外則是獨具中國特色的中國版教材。我前一段時間看了一本中國版的消費者行為學，所謂中國版的涵義就是增加了獨具中國特色的案例，另外對一些晦澀或者無法精準翻譯的內容替換掉，非常遺憾的是限於譯者水平的限制，我發現了非常明顯的錯誤，而且明顯不止一處。大概認為是沒人會閒著去對照英漢兩本教材去看究竟刪除或者篡改了那些內容。

    所以這就是會一門外語的用處。很遺憾，我至今仍然是半瓶水晃盪，無法達到流利的應用。大概唯一的好處，就是勉強能夠應付自己的使用，強推一本書，大部分時候是夠了，如果碰上非常殘忍的大部頭，表示無能為力。不過坦白講，大部分的原版教材，尤其是能夠再版到11th，12th這種類型的書，其實是經過市場的考驗的，大部分有頑強的生命力，這跟dictionary其實是一樣的道理，必須要編者或者著者有足夠的耐心，耐得住寂寞。我判斷好書的標準很簡單，就是這本書能夠衍生出來多少條道路，或者曾經引用了的文獻，雖然這個方式有些粗暴，但是有效。從critical thinking的角度去看，沒有足夠的證據證明的conclusion不具備被相信的價值。為知識付費啊，現在大部分的人有焦慮，每天聽著各種精心編排的資訊，快速的瀏覽了這種資訊，減緩了焦慮，獲得了知識的快感。所有的知識獲取都不是簡單的瀏覽一遍就可以儲存的，每個觀點，每個片段都需要經歷反覆的閱讀，切割，匹配，繫結與固化，最終成為你memory中非常重要的一部分，然後當你某天觸發了keyword，再快速的從記憶中喚起儲存的片段。這大概是電子書的唯一好處，如果掌握的好，可以快速的進行有效的主題閱讀。

    說一千道一萬，到了後續階段，還是需要你快速的進行實踐。從資訊變成知識，到最後變成技能需要你痛苦的實踐過程。所以實際上講，所有的路徑最後只有可選的兩條，一種是固化的知識去汲取，不管是書本，還是視訊或者音訊也好，最終只是手段；另外一條路徑，則是直接跟行業內的人快速交流，歸根到底必須跟一線的實操人員進行接觸，而不是多少年前做過的人溝通。誠然，跟最後走上管理崗的人交流，你可以獲得開闊的視野，思考問題的角度最終還要受到你認知框架的限制，但是從另外一個角度講，到了管理崗之後實質上很多會脫離一線，脫離之後其認知很難被快速的更新。當然，這些是現在培訓機構的通常套路，從某家知名機構出來走穴的，跟培訓機構合作大家分賬。被請上講臺的人，從某種程度上講，已經自帶了主角模板，如果不犯特別大的錯誤，基本上可以保證有一定體面的生活。

    當然，這是在討論讀書，所以也沒必要扯開話題太多，再重新歸納一下，其實也只是一點，盯著某個學科的一本具有脈絡指導性質的書，先翻爛了再說後面的延伸。
